Field Papers autour de Gérardmer


Un carnet de FieldPapers donne un fond de cartes pour faire des relevés sur le terrain : il faut les imprimer pour pouvoir les emmener avec soi. En se promenant, on y marque au crayon ce qu'on peut observer dans le paysage, (on peut dans le même temps faire des photos géolocalisées, prendre une trace gpx). C'est la phase de collecte.
De retour chez soi, on se sert ensuite de tous ces éléments pour renseigner la base de données OpenStreetMap. On utilise alors un éditeur spécialisé : ID ou JOSM. C'est la phase de saisie.
